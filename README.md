# Raynos dotfiles

## Installation

### Clone

```sh
cd ~/projects
git clone git@github.com:Raynos/dotfiles
```

### Install (optional)

**UBUNTU ONLY**

If your running ubuntu you can run

```sh
bash install.sh
```

to install all the dependencies

### Bootstrap

To copy all the files from `~/projects/dotfiles` into `~`

```sh
source bootstrap.sh
```
